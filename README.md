# Aether: The Student Resiliency Ecosystem

Aether is a next-generation, research-driven platform for student mental health resilience, built on a modular, privacy-first, and highly maintainable architecture. It leverages multi-modal AI, federated learning, and a human-in-the-loop crisis intervention model. This project is inspired by the 2026 research proposal and is designed to be generic, visually appealing, and responsive across devices.

## Features
- **Echo Chamber**: Anonymized, voice-enabled outlet for catharsis with NLP-based crisis detection.
- **AI-Triage & Sentiment Mapping**: Real-time linguistic and biometric analysis for wellness mapping.
- **Peer-Navigator Network**: Peer matching for culturally responsive support.
- **Privacy & Data Ethics**: Federated learning, zero-knowledge proofs, and SAFE-AI compliance.
- **Resilience Pathway**: Guided user journey from catharsis to stabilization.
- **Accessibility**: Fully responsive and mobile-friendly design.

## Architecture
- **Monorepo**: Modular structure with separate frontend, backend, and shared packages.
- **Next.js**: Modern, scalable web framework for the frontend.
- **API-Ready Backend**: For AI, biometric, and peer-matching services.
- **Shared UI**: Reusable, accessible, and themeable components.
- **Site Config**: Centralized configuration and presets.

## Getting Started
1. **Install dependencies**
   ```bash
   cd aether
   npm install
   ```
2. **Run the development server**
   ```bash
   npm run dev
   ```
3. **Open the app**
   Visit [http://localhost:3000](http://localhost:3000) in your browser.
